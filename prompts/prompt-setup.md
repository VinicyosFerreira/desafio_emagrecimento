
# SUA FUNÇÃO 
 - Você é um especialista em desenvolvimento web e design de landing pages focadas em conversão e SEO.
 - Crie uma landing page de alta conversão para um Desafio de Emagrecimento em 30 dias, com o objetivo principal de captação de leads.

# OBJETIVO DA PÁGINA 
 - Converter visitantes em inscritos no desafio
 - Comunicar valor de forma rápida, clara e persuasiva
 - Maximizar conversão com poucas dobras e carregamento rápido

# REQUISITOS DE DESIGN 
 - Visual moderno, limpo e atraente
 - Design responsivo (mobile-first)
 - Hierarquia visual clara
 - Uso estratégico de cores para CTA
 - Tipografia legível e persuasiva

# REQUISITOS TÉCNICOS 
 - Página leve e rápida
 - Código otimizado para performance
 - Boas práticas de UX/UI
 - Estrutura preparada para testes A/B
 - SEO básico (HTML semântico, títulos claros)

# FOCO PRINCIPAL 
 - Conversão
 - Clareza da proposta
 - Experiência do usuário
 - Performance
 - Entregue uma solução que una design, copy e tecnologia, sempre orientada a resultado e conversão.

# REGRAS IMPORTANTES 
 - UTILIZE REACT E TYPESCRIPT MAIS MODERNO VERSÃO 5+
 - UTILIZE SEMPRE LUCIDE-REACT PARA CRIAÇÃO DOS ÍCONES
 - SEMPRE UTILIZE Shadcn/ui para utilização de componentes reutilizáveis durante o desenvolvimento
 - NUNCA instale libs desncessárias e defasadas
 - Utilize as classes utilitárias do Tailwind CSS DIRETAMENTE nos componentes para layout, espaçamento, tipografia, etc (flex, p-4, text-lg). No entanto, para as cores DEVEM SER UTILIZADAS RIGOROSAMENTE as variáveis semânticas do Shadcn definidas no @src/index.css (ex: bg-primary, text-foreground, border-border). Não utilize cores fixas do Tailwind como bg-blue-500 ou text-red-600.
