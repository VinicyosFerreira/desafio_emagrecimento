# Crie um um sistema de roteamento com React Router Dom 

## Regras
- Utilize as melhores práticas do React moderno para criação de rotas 
- Crie as rotas abaixo
- Crie em @src uma pasta pages contendo as páginas com mesmo nome das rotas definidas abaixo
- Reutilize o Header e o Footer do projeto em todas as páginas no lugar adequado

## Rotas
- / -> Página inicial
- /about -> Página sobre
- /benefits -> Página de benefícios
- /testimonials -> Página de depoimentos

## Home Page
- Insira em HomePage os componentes com Hero.tsx, CardSection.tsx e VideoSection.tsx
- Abaixo de VideoSection e acima de Footer, crie um componente com titulo para Perguntas Frequentes
- Utilize o componente Accordion do Shadcn/ui para criar as perguntas frequentes
- Crie entre 5 a 8 perguntas frequentes com suas respectivas respostas de forma clara e concisa

## About Page
- Crie um título com o texto "Sobre"
- Crie 2 a 3 parágrafos completos sobre o desafio de emagrecimento com 30 dias em portugues do Brasil
